1.ntroduction

My research question is s total amount of debit and credit card expenditure related to consumer price index, unemployement rate and weighted average interest rates for banks loans? The reason that I choose this topic is nowadays most of the time people use debit or credit cards instead of cashes. The idea behind this question is try to figure out which factors can affect percentage of credit or debit card usage. Without any analysis, first of all, I guess that people may be more likely to use cards when the unemployment rate increases. Because they need to maintain some basic needs. In addition, when the interest rate increases, people can decrease their habit of using cards. Maybe, they consider that usage of cards are not that much profitable at this point. Another guess is that people will increase their card usage due to the price index increase. Due to the price increases, they may not be able to pay cash to buy some products, so there might be a tendency to use cards.

2. Analysis Steps

Firstly, I decided of my data sets and dependent statistics. Each of my data set comes from different categories and their frequency is monthly. Then I downloaded these data sets from https://evds2.tcmb.gov.tr/ as an Excell file. After that I started to use Rstudio and I install the packages which I would have need in the analysis.

install.packages(data.table) library(data.table) install.packages(xlsx) install.packages(zoo) install.packages(xts) install.packages(ggpubr)

library(zoo) library(xts) library(readxl) library(ggpubr)

Then, read the Excel files by using the following codes.office online

data <- read_excel(C:/Users/LENOVO/Desktop/EVDS.xlsx) View(data)

The statistics that I had include 37 months of observations of my variables. So I created sequences of dates in order to obtain time series object. I created xts object that contains all data.

dates<- seq(from=as.Date(2016-10-01),to=as.Date(2019-10-31),by=months) genelxts<-xts(x=data[1:37,],order.by=dates)

The statistic that I wanted to interpret the relationsahip with other variables is Amount of credit and debit card expenditure. I created another time series object for this statistics.

card_exp <- data[1:37,2]
cardxts<- xts(x=card_exp,order.by=dates) storage.mode(cardxts) <- numeric str(cardxts)

I wanted to see the relationships with Card expenditure with three variables. First of them is average interest for banks loans. I created xts object for this variable

loan_interest<- data[1:37,3]
loanxts<- xts(x=loan_interest,order.by=dates) #xts object for interest rates of loans storage.mode(loanxts)<- numeric

The second variable is consumer price index. So I created another xts object for consumer price index.

price_index <- data[1:37,4]
pricexts <- xts(x=price_index,order.by=dates) #xts object for price indexes storage.mode(pricexts)<- numeric

The third one is unemployemt rate. I created another xts object for it.

unemp_rate <-data[1:37,5] unempxts <- xts(x=unemp_rate,order.by=dates) #xts object for unemployement rates storage.mode(unempxts)<- numeric

I have created 5 xts object. As I mentioned the first one is combination of all xts objects.Then I plot the 4 of my xts object to visualize.

plot(cardxts)
plot(loanxts) plot(pricexts) plot(unempxts)

Then I merge the other xts objects with my card expenditure xts object

merge(cardxts,loanxts) merge(cardxts,pricexts) merge(cardxts,unempxts)

I plot my dependent variable card expenditure xts object with other xts object in order to visiualize their relationship.

qqplot(loanxts,cardxts) #qqplots of cardxts against other xts objects qqplot(pricexts,cardxts) qqplot(unempxts,cardxts)

I applied correlation test to find the correlation coefficient and by using it analysize the relationship of card expenditure with other variables.

cor1 <- cor.test(cardxts, loanxts, method = pearson) #correlation measurments between cardxts and other xts objects cor2 <- cor.test(cardxts, pricexts, method = pearson) cor3 <- cor.test(cardxts, unempxts, method = pearson)

Finally I want to check the periodicity of my xts object.

periodicity(cardxts) periodicity(loanxts) periodicity(pricexts) periodicity(unempxts)
